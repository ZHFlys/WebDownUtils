Web批量下载助手

版本 1.0.0

✨ 主要功能

  • 🌐 整页下载 - 一键下载页面所有媒体文件
  • 🔲 区域选择 - 精确选择特定区域下载
  • 🔍 智能筛选 - 多级筛选系统，快速定位文件
  • 🎵 音频支持 - 全面支持MP3、WAV、FLAC等音频格式
  • 📊 智能排序 - 按时间、大小、名称、类型排序
  • 🔄 网络监听 - 实时捕获动态加载的媒体文件

🔧 技术特性

  • 基于 Manifest V3 的现代化扩展
  • 支持页面扫描和网络监听双重获取
  • 智能文件去重和大小估算
  • 响应式设计，适配不同屏幕
  • 本地处理，保护用户隐私
